What is Css

  1. Css stands for Cascading Style Sheet
  2. CSS describe how HTML element should display on Screen, paper or in other media
  3. CSS is not neccesary it saves lots of work it can control the layout of multiple webpage all at once

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribite inside the HTML elements.
  • Internal - by using the style elements in the head section.
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS is to keep the styles in the external CSS files. However, In this tuitorial we will use inline and internal styles, because this is easy to demonstrate and easiar for you to try it yourself.

When comparing CSS class vs ID the diffrence is that CSS class applies a style tp multiple elements. ID , on other hand , applies a style to one unique elements. ID is special in that you can use special URL to link directly to an element and it's by Javascript.

In CSS, selectors are used to target a specific element or range of elements om web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should applied.
